# Presentation

Files for the final presentation live here.

## Main Files
- `generate_presentation.py` — generates the PowerPoint deck
- `NET3006A_ML_Network_Telemetry_Presentation.pptx` — generated presentation file
- `presentation-outline.md` — presentation flow and speaking order
- `presenter-summary-plain-language.txt` — simple-language presenter summary

## Purpose

This folder is meant to be easy to scan quickly on GitHub:
- the generator creates the deck
- the `.pptx` is the final output
- the outline helps organize delivery
- the plain-language summary helps presenters explain the topic clearly
