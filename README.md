# NET3006 Project Repository

This repository contains coursework for `NET3006A — Network Management and Machine Learning`.

## Main Project

The primary project in this repository is:

- `NET3006A-ML-Network-Telemetry/` — a survey project on machine learning for network telemetry

## Project Topic

The topic of the main project is **Machine Learning for Network Telemetry**.

It looks at how machine learning can help networks:

- detect unusual behavior
- predict performance problems
- improve quality of service
- support smarter automation

## Repository Layout

- `NET3006A-ML-Network-Telemetry/` — main coursework project
- `README.md` — repository overview
- `.gitignore` — local files that should not be committed

## Key Project Areas

Inside `NET3006A-ML-Network-Telemetry/`:

- `presentation/` — deck generator, PowerPoint, speaking outline, and presenter summary
- `notes/` — member-specific reading notes
- `synthesis/` — shared research summaries and open challenges
- `references/` — bibliography files
- `proposal/` — proposal generator

## How To Regenerate The Presentation

From the project folder:

```bash
cd "NET3006A-ML-Network-Telemetry/presentation"
python3 generate_presentation.py
```
